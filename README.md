# Blade

A third-person melee combat vertical slice built with Unreal Engine 5.8 and the Gameplay Ability System (GAS). Work in progress — this description will be updated as the project evolves.

## Project Features

- Third-person Melee Character Movement
- Enhanced Input
- Orient-to-movement locomotion
- Walk / Run with speeds matched to animation
- Jump, fall, and landing animation states
- Locomotion Blend Space
- Cached pose for upper-body combat animations
- Custom AnimInstance in C++ (shared by player and AI)
- Gameplay Ability System foundation
- AttributeSet (Health and Posture — no stamina)
- Shared character base for player and AI
- Mix of C++ & Blueprint
- Sword combat animation set
